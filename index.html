Available Variables

  • [-email-] : Recipient Email
  • [-time-] : Date & Time
  • [-emailuser-] : Email User (user@domain)
  • [-randomstring-] : Random alphanumeric string
  • [-randomnumber-] : Random number
  • [-randomletters-] : Random letters
  • [-randommd5-] : Random MD5 hash

Example

Recipient: user@domain.com

  • hello [-emailuser-] → hello user
  • your code is [-randommd5-] → e10adc3949ba59abbe56e057f20f883e
